# Products Manager (Учебный проект)

Простой учебный проект на Python для работы с товарами и их ценами.  
Программа позволяет:

- проверять цену существующих товаров;
- добавлять новые товары с ценой;
- сохранять данные в файл `products.json`.

## Использование

1. Запустите скрипт:
   ```bash
   python products.py

2. Введите название товара:
- Если товар есть - выведет цену
- если нет - предложит ввести цену и сохранит товар в products.json
